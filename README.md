# About crawler
enter a website then it will try to get the screenshot, title, description, and body of the website.

# Prepare
```
dos2unix init.sh
chmod 777 init.sh
./init.sh env
```

# Run
## frontend
```
sodu npm run dev
```
## backend
```
sudo php artisan serve
```
